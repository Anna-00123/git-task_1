# Git_task_1

Навчальний репозиторій для практичної роботи «Git і GitHub. Git flow».

## Що зроблено

- Створено локальний репозиторій та налаштовано користувача Git.
- У гілку master додано декілька комітів (file1.txt, file2.txt, file3.txt).
- Продемонстровано перегляд історії комітів та переключення між ними.
- Створено гілку develop, у неї додано коміти (devA.txt, devB.txt).
- Від develop створено гілку my-feature, додано коміти (feature1.txt, feature2.txt).
- Гілку my-feature злито у develop, після чого видалено.
- Гілку develop злито у master, після чого видалено.
- Створено віддалений репозиторій на GitHub, відправлено вміст локального репозиторію.
- Продемонстровано клонування репозиторію, коміт у клоні та pull оновлень.
